# サーモンラン記録ウェブページ

## プロジェクト概要
- サーモンラン（スプラトゥーン）のプレイ記録・統計を管理するウェブページ
- 個人利用を想定

## 技術構成
- 素のHTML/CSS/JavaScript（フレームワークなし）
- データ保存: localStorage（ブラウザ内）
- テーマ: ダークテーマ（オレンジ × ネイビー）

## ファイル構成
```
salmonrun_webpage/
├── index.html          ← メインページ
├── css/style.css       ← スタイル
├── js/app.js           ← データ処理・表示ロジック
├── data/records.json   ← データ用（将来利用）
└── CLAUDE.md           ← このファイル（開発方針・決めごと）
```

## 現在の機能
- 記録入力フォーム（日付・ステージ・キケン度・金イクラ・結果）
- 記録一覧テーブル（新しい順）
- 統計表示（総バイト数・クリア率・最高/平均金イクラ）
